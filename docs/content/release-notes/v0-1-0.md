First release of `dhub`, a Go CLI for browsing Docker Hub public data.

No authentication required. All commands read from the public Docker Hub v2 API.

## Commands

- `dhub search <query>` — search Docker Hub images; `--official` and `--automated` filters
- `dhub image <name>` — show metadata for a single image (`nginx` or `user/repo`)
- `dhub tags <name>` — list tags for an image
- `dhub official` — list Docker Official Images
- `dhub user <username>` — list public repos for a user or organisation

## Output formats

`table`, `json`, `jsonl`, `csv`, `tsv`, `url`, `raw`. Auto-detects TTY.

## Smoke test

```sh
dhub search nginx -n 3 -o table
dhub image nginx
dhub tags redis -n 5
dhub official -n 10
dhub user bitnami -n 5
```
