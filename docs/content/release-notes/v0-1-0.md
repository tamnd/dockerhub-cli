First public release. A single Go binary that reads Docker Hub public data
through the v2 API. No account, no API key, no setup beyond downloading the
binary.

## Commands

| Command | What it does |
|---------|-------------|
| `dhub search <query>` | Search repositories. `--official` and `--automated` narrow results; `-n` caps the count. |
| `dhub image <name>` | Full metadata for one image (`nginx` or `user/repo`). |
| `dhub tags <image>` | Tags with digest, OS, architecture, and compressed size. |
| `dhub official` | Browse the Docker Official Images library namespace. |
| `dhub user <username>` | Public repositories for a user or organisation. |
| `dhub version` | Print version, commit, build date, and Go runtime. |

## Output formats

Pass `-o` to choose: `table` (default on a terminal), `json`, `jsonl`
(default when piped), `csv`, `tsv`, `url`, or `raw`. Use `--fields` to select
columns and `--template` for a Go text/template applied per record.

## Distribution

Linux, macOS, Windows, and FreeBSD archives on the GitHub release page.
Linux deb, rpm, and apk packages. Multi-arch container image on GHCR:

```
ghcr.io/tamnd/dhub:0.1.0
```

## Quick check

```sh
dhub search nginx -n 3 -o table
dhub image nginx
dhub tags redis -n 5
dhub official -n 10
dhub user bitnami -n 5
```
